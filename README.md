# ETF Penetration Analysis Tool

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

📊 Analyze ETF underlying holdings with one click

## Quick Start

```bash
cd backend
pip3 install -r requirements.txt
python3 app.py
```

Visit: http://localhost:5001

## Features

- ✅ Auto-detect ETFs in portfolio
- ✅ Calculate penetrated holdings
- ✅ Smart industry classification (1800+ stocks)
- ✅ Privacy mode (hide amounts)
- ✅ Export to Excel
